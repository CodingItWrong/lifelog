# Lifelog

A minimal attempt at tracking visits to places in a privately-owned app.

## Getting Started

### Requirements

1. Ruby
1. PostgreSQL (e.g. [Postgres.app][postgres-app])

### Setup

```sh
$ bin/bootstrap
```

### Testing

```sh
$ bin/rspec
```

### Running

```sh
$ bin/rails server
```

[postgres-app]: http://postgresapp.com
